# Fake Website Detector

A Python + Flask project to detect whether a given website is **fake or real** using ML-based features and heuristics.

1. Clone the repository:
   ```bash
   git clone https://github.com/Mdamanatansari7/fake-website-detector.git
   cd fake-website-detector
   ```

2. Create a virtual environment:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate   # Windows
   source .venv/bin/activate  # macOS/Linux
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Run the app:
   ```bash
   python app.py
   ```

5. Open browser and visit:
   ```
   http://127.0.0.1:5000/
   ```
